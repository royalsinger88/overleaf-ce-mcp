# overleaf-ce-mcp 快速上手（10 条命令）

适用场景：
- 你已经有自建 Overleaf CE 地址。
- 你在 VPS 上操作，认证文件用 `/root/.olauth`。
- 目标是最快跑通“本地稿件 -> 上传 CE -> 编译验证”。

示例变量（先改成你的）：

```bash
export CE_URL="http://129.150.32.65:17880"
export AUTH_FILE="/root/.olauth"
export WORK_DIR="/root/overleaf-workspace"
export PROJ_DIR="$WORK_DIR/test"
export PROJ_NAME="test"
# 可选：提高 Semantic Scholar 检索稳定性
export S2_API_KEY="your_semantic_scholar_key"
```

## 6. 更新已有项目（推荐：merge）

```bash
python - <<'PY'
import os
import tempfile, zipfile, shutil
from pathlib import Path
from overleaf_ce_mcp.upload import package_project_for_upload
from overleaf_ce_mcp.sync import ols_sync

pack = package_project_for_upload(project_dir=os.environ["PROJ_DIR"])
stage_root = Path(tempfile.mkdtemp(prefix="oe-stage-")).resolve()
stage_dir = stage_root / os.environ["PROJ_NAME"]
stage_dir.mkdir(parents=True, exist_ok=True)
with zipfile.ZipFile(pack["zip_path"], "r") as zf:
    zf.extractall(stage_dir)

(stage_dir / ".olignore").write_text(
    "*.aux\n*.bbl\n*.blg\n*.fdb_latexmk\n*.fls\n*.log\n*.out\n*.spl\n*.synctex.gz\n*.pdf\n*.zip\n",
    encoding="utf-8",
)

code, out, err = ols_sync(
    workspace_path=str(stage_dir),
    mode="local-only",
    project_name=os.environ["PROJ_NAME"],
    ce_url=os.environ["CE_URL"],
    store_path=os.environ["AUTH_FILE"],
    olignore=str(stage_dir / ".olignore"),
    delete_policy="i",  # merge: 忽略远端多余文件
    verbose=False,
)
print("exit_code:", code)
print(out[-1200:])
print(err[-600:])
shutil.rmtree(stage_root, ignore_errors=True)
PY
```

## 12. 常用诊断命令

```bash
# 查看项目文件树（远端）
python - <<'PY'
import os
import pickle
from olsync.olclient import OverleafClient
auth = pickle.load(open("/root/.olauth", "rb"))
cli = OverleafClient(cookie=auth["cookie"], csrf=auth.get("csrf"), ce_url=os.environ["CE_URL"])
proj = cli.get_project(os.environ["PROJ_NAME"])
info = cli.get_project_infos(proj["id"])
root = info.get("project", info)["rootFolder"][0]
paths = []
def walk(folder, prefix=""):
    for d in folder.get("docs", []):
        paths.append(prefix + d["name"])
    for f in folder.get("fileRefs", []):
        paths.append(prefix + f["name"])
    for sub in folder.get("folders", []):
        walk(sub, prefix + sub["name"] + "/")
walk(root, "")
for p in sorted(paths):
    print(p)
PY
```

## 13. 最小排错清单

- `projects: 0` 或直接异常：先检查 `.olauth` 是否过期。
- `compile status=failure`：看返回里的 `response_excerpt`，再看项目源文件。
- 同步后正文没上去：检查 `.olignore` 是否误包含 `sections/*` 或 `references.bib`。
- 更新错项目：确保 `PROJ_NAME` 与远端项目名完全一致。
